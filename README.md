# 🔬 SCIENCE PULSE

A premium bilingual science news platform built with **Angular 17**, featuring a stunning **Liquid Glass** dark UI.

## 🚀 Quick Start

```bash
npm install
npm start
# → http://localhost:4200
```

## Features
- Bilingual Georgian/English with instant switching
- Liquid Glass dark space UI with animations
- Posts with categories, likes/dislikes, comments  
- Interactive quizzes with scoring
- Role-based auth (user / creator / admin)
- Image upload (Base64)
- Auto hero slider from latest 5 posts
- Admin dashboard, user management
- Full localStorage persistence — no backend needed

## User Roles
- **user**: read, like/dislike, comment, take quizzes
- **creator**: + create posts & quizzes
- **admin**: + manage users, delete comments

## Tech Stack
Angular 17 · TypeScript · SCSS · Standalone Components · Signals · LocalStorage
